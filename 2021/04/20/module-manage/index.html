前端模块化

模块化的开发方式可以提供代码复用率，方便代码管理，一般一个文件就是一个模块，有自己的作用域，只向外暴露特定的变量和函数。

ES6之前，原生不支持模块化，所以出现了第三方的解决方案，分别是AMD与CMD，NodeJs诞生后使用CommonJs的模块化规范。

AMD

异步模块定义Asynchronous Module Definition，是RequireJs在推广过程中对模块定义的规范化的产出。

CMD

通用模块加载规范Common Module Definition，是Sea.js实现的规范。
